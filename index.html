POSH-HPOneView

This library provides a pure Windows PowerShell interface to the HP OneView REST APIs.

HP OneView is a fresh approach to converged infrastructure management, inspired by the way you expect to work, with a single integrated view of your IT infrastructure.

This PowerShell project is developed for those that want to automate operations or use PowerShell as an interactive CLI for HP OneView.

To get started, you can download the latest release in an easy to use, digitally signed installer.

Alternatively, you can clone the source repository:

[PS] C:\> cd <repo_directory>
[PS] C:\> #Clone POSH-HPOneView repository
[PS] C:\> git clone https://github.com/HewlettPackard/POSH-HPOneView.git
